## User Flow Examples

### Example 1: Running Network Diagnostics
1. Click **Diagnostics** tab
2. Enter host (or use default 8.8.8.8)
3. Click **Run Ping Test**
4. View results in text box below
5. Optionally run DNS lookup or traceroute

### Example 2: Fixing Security Issues
1. Click **Security** tab
2. Click **Run Vulnerability Scan**
3. Review detected issues
4. Click **View Available Fixes**
5. Click specific fix button (e.g., **Enable Defender**)
6. Confirm in dialog
7. View execution results

### Example 3: Profiling Performance
1. Click **Monitoring** tab
2. View real-time stats
3. Click **Run Performance Profile**
4. Wait 5-10 seconds
5. View detailed report in popup
6. Check bottlenecks and recommendations
7. Close popup when done
